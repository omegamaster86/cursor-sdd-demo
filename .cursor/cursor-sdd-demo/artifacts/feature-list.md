# 機能一覧（cursor-sdd-demo）

**目的**: プロダクトのスコープ（MVP/Phase）と、UI/API/バッチの一覧を"合意の一次ソース"として固定する。

---

## 画面機能

| 機能 ID | 画面名 | 説明 | 優先度 | Phase | 記載者 |
| ------- | ------ | ---- | ------ | ----- | ------ |
| UI-001  | アンケート一覧 | 作成済みアンケートの一覧表示 | High | MVP | |
| UI-002  | アンケート作成/編集 | アンケートの新規作成・編集フォーム | High | MVP | |
| UI-003  | 回答結果一覧 | アンケートの回答者一覧・回答数表示 | High | MVP | |
| UI-004  | 回答詳細 | 個別回答者の回答内容表示 | High | MVP | |
| UI-005  | お知らせ一覧 | お知らせの一覧表示 | High | MVP | |
| UI-006  | お知らせ作成/編集 | お知らせの新規作成・編集フォーム | High | MVP | |

---

## API 機能

| 機能 ID | API 名 | エンドポイント | メソッド | 説明 | レベル | 優先度 | Phase |
| ------- | ------ | -------------- | -------- | ---- | ------ | ------ | ----- |
| API-001 | アンケート一覧取得 | /api/surveys | GET | アンケート一覧を取得 | B | High | MVP |
| API-002 | アンケート作成 | /api/surveys | POST | 新規アンケートを作成 | B | High | MVP |
| API-003 | アンケート詳細取得 | /api/surveys/:id | GET | アンケート詳細を取得 | B | High | MVP |
| API-004 | アンケート更新 | /api/surveys/:id | PUT | アンケートを更新 | B | High | MVP |
| API-005 | 回答一覧取得 | /api/surveys/:id/responses | GET | 回答者一覧を取得 | B | High | MVP |
| API-006 | 回答詳細取得 | /api/surveys/:id/responses/:rid | GET | 個別回答内容を取得 | B | High | MVP |
| API-007 | お知らせ一覧取得 | /api/notices | GET | お知らせ一覧を取得 | C | High | MVP |
| API-008 | お知らせ作成 | /api/notices | POST | 新規お知らせを作成 | C | High | MVP |
| API-009 | お知らせ詳細取得 | /api/notices/:id | GET | お知らせ詳細を取得 | C | High | MVP |
| API-010 | お知らせ更新 | /api/notices/:id | PUT | お知らせを更新 | C | High | MVP |

**レベル定義**:
- **B**: 複数テーブル結合/集計/加工あり
- **C**: 単テーブルCRUD相当
